# 4. 错误码

若商户请求AllPay网关时，请求的参数错误或参数缺失时，AllPay网关会将错误信息以JSON格式返回给客户端，供商户解析调用，以下是相关错误信息返回参数的说明

参数应答码code对应应答信息msg
U1Clientid not found
U2Signature error
V1Parameter missing
E1Exceed the limit
R1Orderno repeat
C1Order currency is not set
P1paychannel error
I1Illegal request address
T1Parameter missing（version或tranType）
T2Txntype error
T3transaction is not exist or fail
T4The maximum amount exceeds the upper limit
99System error
